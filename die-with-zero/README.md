## 设计哲学

- **AI 自适应章节数**——原书 9 章 + Conclusion，深读教材压缩成 6 章（每章 maps 到 1-2 章的概念簇）
- **学术引用白名单**——只用 Bill 自己的 bibliography（PDF Notes 提取）+ web search 的当代解读，**不编造引用**
- **挑战式语调**——每章至少一处 pullbox.warn 推回读者的"我懂了"
- **读者门槛诚实**——明确指出这本书<strong>不适合</strong>所有人（under-savers 不该照搬）
- **包含反对声音**——White Coat Investor / Bogleheads / Accidentally Retired 等 critique 在 ch1 / ch6 引用

## 怎么用

1. **从 [index.html](index.html) 的能力金字塔开始**——看清这本书要在你身上建什么
2. **做 §4 的 self-assessment**——24 个 checkbox，诚实勾。决定你目前在第几层
3. **按推荐路径读**（§6 4-6 周分 Phase）——不要快速过一遍
4. **每章读完<strong>立即</strong>做反思练习**——光读不写 → 80% 价值流失
5. **6 个月后回来重读 §7 mastery test**——验证行为有可观察改变

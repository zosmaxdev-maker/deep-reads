# deep-reads

Multi-book deep-read tutorial collection. Each book turned into a structured HTML self-study package with academic citations, reflection exercises, self-check quizzes, and a capability map.

→ **Live site: [https://zosmaxdev-maker.github.io/deep-reads/](https://zosmaxdev-maker.github.io/deep-reads/)**

Generated by the [`book-to-deep-reads`](https://github.com/zosmaxdev-maker/productive) skill.

## Books

| Book | Author | Chapters | Entry |
|---|---|---|---|
| Die With Zero | Bill Perkins (2020) | 6 | [die-with-zero/index.html](die-with-zero/index.html) |
| The Psychology of Money | Morgan Housel (2020) | 6 | [the-psychology-of-money/index.html](the-psychology-of-money/index.html) |

Each book has:
- `index.html` — capability map (capability pyramid + self-assessment + failure paths + learning path)
- `outline.html` — chapter-by-chapter table of contents
- `ch{1..N}-<topic>.html` — chapter tutorials
- `_research/sources.md` — citation whitelist
- `_research/strategy.md` — chapter plan
- `deploy.sh` — optional standalone deploy script

## Privacy

- All HTML files carry `<meta name="robots" content="noindex, nofollow, noarchive">`
- Site-level `robots.txt` blocks all crawlers
- `.nojekyll` lets GitHub Pages serve `_research/` paths

Anyone with the URL can view — but search engines won't index.

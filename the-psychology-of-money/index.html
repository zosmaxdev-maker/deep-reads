§0 · The Real Goal

读这本书的真正目的

多数读《金钱心理学》的人会得到一堆名言警句——Buffett 说、Munger 说、运气和风险是双胞胎——然后什么都没改变。原因不是书写得不好，是读者把它当作"知识"读，而它是按"能力训练手册"写的。差别如下：

把它当知识读把它当能力建
问的问题这章讲了什么？这章在我身上长什么能力？
验证能复述书里的观点能在压力下做出对应的行为
测试朋友聊天时引用一句市场 -30% 时你的实际动作
结果下次读到类似书有"似曾相识"portfolio 行为有可观察的改变

这一页的承诺

读完这一页，你应该能用能力清单（不是章节清单）回答："我目前在第几层？我下一步要建造什么？""我没有这个能力会怎么死？"——如果做不到，整本书等于白读。

关键关系

L1-L2 是认知层（看清世界和自己）；L3-L4 是价值层（决定要什么、要守什么）；L5-L6 是执行层（在压力下不偏航 + 系统化）。三对各有不同的训练方式：认知层靠读 + 反思；价值层靠写下来 + 时间检验；执行层靠规则化 + 复盘。

§2 · The Six

6 种能力详解

每一项给：定义 + 已建立的信号（你看到这些说明它在你身上 working）+ 没建立的危险信号（这些出现说明该层还没盖好）。

L1 · C1

自我考古

Self-Archaeology → Ch.1

看清你自己的金钱观从哪来——cohort 经历、家庭叙事、时代背景——并接受智力不是金融游戏的入场券，行为才是。

已建立 ✓

  • 能列出 3 个你的"金钱默认设置"和它们的来源
  • 看到聪明人破产不再吃惊
  • 对 7 个认知 bug 承认"我也有"，不再说"我知道所以我不会"
  • 评价别人的金钱决定时先问 cohort，不再急着判对错

未建立 ✗

  • 仍认为"我比一般人聪明所以我能跑赢"
  • 觉得"读了这书我已经免疫 loss aversion"
  • 认为"我所在的群体 / 同事 / 工程师圈"的金钱观是"理性的默认"
  • 把 Buffett / Munger 的方法直接套用而不问 cohort
L2 · C2

归因清醒

Attribution Sobriety → Ch.2

把运气、风险、时间、长尾分布作为一阶变量进入归因模型——不再把单次结果当成技能证据，不再用"年化收益率"作为策略主指标。

已建立 ✓

  • 评价投资决策时分别看 process 和 outcome（不 resulting）
  • 策略主 KPI 是"持续 30 年不中断"，不是"打败 S&P 几个点"
  • 承认 4% 决定 100%——核心仓位用宽基指数
  • 不羡慕"赚得快"的人，因为知道方差含义

未建立 ✗

  • "我朋友买 X 翻倍了我也跟"
  • 仍在追求年化"再高 2 个点"的优化
  • 对长尾分布的概念可以复述但行为上仍假设高斯
  • 看到自己一笔投资盈利 3x 就当成 process 验证
L3 · C3

生存优先

Survival First → Ch.3

承认致富与守富是两套不同的能力——致富要乐观冒险，守富要悲观谦逊。安全边际不是"浪费效率"，是续航能力。给"够了"画一个具体数字。

已建立 ✓

  • 致富仓 / 守富仓分账户，不允许跨账户调用
  • 有写下来的应急金规则（金额 + 触发情境 + 复盘节奏）
  • 知道你的"够了"数字（净资产 / 被动收入 / 自由月数）
  • 看到 LTCM / Madoff 案例不再"我不会犯那种错"——而是"什么机制能保证我不犯"

未建立 ✗

  • portfolio 里有任何"一夜归零会毁掉你"的部分（高杠杆 / 单股集中 / 流动性差资产）
  • "目标准星"持续上移，永远没有"够"的时刻
  • 把守富等同于"保守不投资"——错把生存当 frozen
  • 没有书面预案应对裁员 / 医疗 / 身份变化
L4 · C4

目的明确

Purpose Clarity → Ch.4

钱真正要买的是对自己时间的控制权——不是物品，不是地位。区分 rich（消费）与 wealthy（积累），并把"自由"翻译成可量化的指标（FU money 数字、自由月数、能拒绝什么工作）。

已建立 ✓

  • 每年算一次"自由月数"（净流动资产 / 月支出）
  • 消费决策时区分"买物品"和"买控制权"
  • 不再羡慕同事的车 / 表 / 房——知道那不是 wealth
  • 有具体的 FU money 数字

未建立 ✗

  • 升 TC 后第一件事是升级生活方式（lifestyle creep）
  • 觉得"自由"是模糊概念无法量化
  • 用"显示性消费"维持自我认同
  • 积累了钱但不知道为了什么——继续打不喜欢的工
L5 · C5

反陷阱防御

Trap Defense → Ch.5

在三个最隐蔽的陷阱前不被 hijack：(a) 数学最优 ≠ 行为可执行；(b) 历史回测漂亮 ≠ 未来不会有黑天鹅；(c) 悲观叙事更易得人信任 ≠ 悲观更准确。

已建立 ✓

  • 策略选"我能拿住的"，不是"Excel 算出最优 Sharpe 的"
  • 有"black swan budget"——假设未来 30 年至少 1-2 次 -50%
  • 实施"媒体节食"——不看每日行情，每周一次摘要
  • 在悲观新闻潮里仍按预定规则定投

未建立 ✗

  • 用回测漂亮的策略加杠杆
  • 因为最近的悲观叙事改变仓位
  • 每天打开 brokerage app 看红绿
  • "这次不一样"——把单次叙事当全样本证据
L6 · C6

系统装配

System Assembly → Ch.6

把 L1-L5 的工具组装成一个你专属的金钱操作系统——明确你玩的是什么游戏；用 Munger 反向思维列"绝不"清单；写下"投资告白"；建立复盘节奏。这是 capstone，没有它前 5 层都白盖。

已建立 ✓

  • 有一份书面的"投资告白" v1+，每年复盘
  • 能在 30 秒内说清你玩的是什么游戏（time horizon / 目标 / 风险）
  • 有明确的"绝不"清单（永不杠杆、单股 ≤ X%、不在 -30% 减仓...）
  • 日历里有定期复盘事件，过去 12 个月真的执行了

未建立 ✗

  • 你的策略只在脑子里，没写下来
  • 每次决策都是"临场判断"
  • 读了书但 portfolio 没改变
  • 从 day trader 那里听到"该减仓"会动摇

§3 · Dependency

能力依赖关系（为什么必须按顺序）

这 6 种能力不是平行的清单。它们有严格的前置关系。

能力需要的前置跳过前置会怎样
L1 自我考古无（地基）—
L2 归因清醒L1没看清自己的 bug，无法识别"我以为是技能其实是运气"
L3 生存优先L1 + L2没认清长尾和黑天鹅，会低估"破产可能"，杠杆变成毒药
L4 目的明确L1 + L3不知道自己 cohort 里"成功"的真定义，会模仿别人的 wealth 错把消费当目标
L5 反陷阱L1 + L2 + L3 + L4没有目的就没有"什么算偏离"，无法识别陷阱
L6 系统装配全部 L1-L5capstone：写出来的"投资告白"是别人的不是你的，压力下崩

最常见的跳层错误

大多数人读完书直接跳到 L6 写"投资告白"——抄一个 60/40，立 5 条规则，然后宣告"我有系统了"。这是假装的 L6。三个月后市场 -20% 的时候这套系统会失效，因为下面 5 层是空的。真正的 L6 是从 L1 一层一层盖上来的副产品，不是直接抄的模板。

评分

  • 2-3 层：你已经有认知层 + 价值层。下一步是 L5 反陷阱——这一层在压力下崩才暴露。

§5 · Common Failures

三种最常见的失败路径

读这本书有三种典型的"读完没用"路径。每一种都是因为绕过了某一层。

失败 A · "我懂了" 综合症（跳过 L1）

读完书觉得"我都懂了"——能复述 7 个认知 bug，能聊 Bessembinder 的 4%——但从未做过 L1 的考古练习，不知道自己的 cohort 默认设置。结果：看到别人贪婪时知道是 bug，看到自己贪婪时仍然认为是"机会"。知识在你身上独立存在，但没接到行为。

解药：停止阅读新书。回到 ch1 的 4 道反思练习，认真写下来。L1 没建好其他层都是空中楼阁。

失败 B · 模板抄袭（跳过 L1-L4 直接到 L6）

常见操作：读了"投资告白"模板，照抄一份 60/40 + 5 条规则，宣告"我有系统了"。这是假装的 L6。压力测试在第一次 -25% 来时——你会发现你"投资告白"里的数字其实是别人的，你拿不住，开始改规则。

解药：承认 L6 是从 L1 一层一层盖上来的副产品。回到 §4 self-assessment 诚实评估。如果 L1-L5 加起来没勾够 12 个，不要写 L6 投资告白——因为它是假的。

失败 C · 守富瘫痪（在 L3 卡住）

读了 ch3 守富的哲学之后，心理上极度向"现金安全"倾斜——把 80% 资产堆在 HYSA，不再投资。把"守富 = 不冒险"等同。这是错的——守富要求你同时能致富，否则通胀会慢慢侵蚀你的购买力。

解药：读 ch4 + ch5——认清"过度合理"和"行为不可执行"的区别。守富是分账户管理，不是单账户全保守。

检查你正在哪一种里

未来 7 天观察自己：(a) 是不是在跟人聊投资时引用书里的金句却没改变自己的 portfolio——你在 A；(b) 是不是写了"投资告白"但没经过 §4 的 12 个 checkbox——你在 B；(c) 是不是 ch3 之后一切都堆现金不投资——你在 C。这三个失败是默认路径，不是少数派。

§6 · Recommended Path

推荐学习路径（4 个 Phase，每 Phase 1-2 周）

不是按章节顺序"快速过一遍"——是按能力构建顺序，每 Phase 都有可验证的产出。

P1

Phase 1 · 认知层（L1+L2）— 1-2 周

读：ch1 行为胜过智能 + ch2 运气与复利。每章必须完成 4 题反思练习并写在 Obsidian 笔记里（Projects/Investing/learning/ 目录）。

产出：一份"我的 cohort 默认设置 + 我的 7 个认知 bug 中最危险的两个" 笔记 + "Process vs Outcome 的 3 个自审记录"。
P2

Phase 2 · 价值层（L3+L4）— 1-2 周

读：ch3 守富的哲学 + ch4 自由与隐形财富。重点输出："我的'够了'数字" + "我的 FU money 数字" + "12-18 个月应急金的计算"。

产出：一份 Projects/Investing/plans/我的财务目标_v0.1.md，包含具体数字和触发情境。
P3

Phase 3 · 执行层（L5）— 1 周

读：ch5 认知陷阱。完成"媒体节食 7 天实验"——不看每日行情，不打开 brokerage app（除 401k 看一次），第 8 天复盘焦虑来源。

产出：一份"我的 black swan budget"——假设未来 30 年至少 1-2 次 -50%，我的 portfolio 在那时是否还可执行。
P4

Phase 4 · 装配（L6）— 1 周（capstone）

读：ch6 你的金钱 OS。完成 §5 的 8 部分"投资告白"模板——每一项至少 30 字，用前 3 个 Phase 的产出作为输入。

产出：Projects/Investing/plans/我的投资告白_v0.1.md + 日历里加好季度复盘事件（1月+7月）。
P5

Phase 5 · 上线（持续）— 6 个月观察

不再读新书。把 v0.1 投资告白真正执行 6 个月。期间观察：(a) 你违反过几条规则？(b) 哪些规则在压力下不可执行？(c) 哪些数字需要修订？6 个月后写 v0.2。

产出：v0.2，并真正用过的 portfolio。这才是这本书真正的 ROI。

§7 · Mastery Test

"已掌握"的 5 个客观判定

判断你是否真的吸收这本书——不是问你能不能复述，是看 5 个具体行为信号。

  1. 你能写出一份让自己 5 年后看也不脸红的"投资告白"。不是抄的模板，是基于你 cohort + 你的"够了"数字 + 你的 FU money 目标定制的。
  2. 你的 portfolio 在 -30% 时不需要你做任何决定。因为决策已经外包给规则——定投继续、再平衡触发、不看 daily。
  3. 你能拒绝高 TC 的 toxic 工作。说明 FU money 概念已经从理论变成杠杆——你的钱给了你拒绝权。
  4. 你不再每天看股价。不是"克制"，是不需要——因为你的系统不依赖你的临场判断。
  5. 你能在 30 秒内回答：如果失业 12 个月你会怎样？因为答案是早就写下的预案，不是临时的恐慌应对。

最简单的 mastery 测试

把这 5 项展示给一个你信任的人，请他随机抽问其中一项。如果你需要思考超过 30 秒——你还在路上。如果你能立刻给出有数字的具体回答——你已经把这本书装进身体了。
